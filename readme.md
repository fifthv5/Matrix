To make the python scripts run when you type a command you can make them aliases
for Bash:
nano ~/.bashrc
then go to the bottom and type
alias matrix='python3 path/to/file'
then save and
source ~/.bashrc
for zsh:
nano ~/.zshrc
then go to the bottom and type
alais matrix='python3 path/to/file'
save and then 
source ~/.zshrc
